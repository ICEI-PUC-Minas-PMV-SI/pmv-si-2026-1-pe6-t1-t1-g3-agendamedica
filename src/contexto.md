# Contexto do Projeto

Este documento descreve o contexto e os objetivos do projeto implementado na pasta `src`.

## Descrição

- Nome do projeto: Agenda Médica
- Propósito: Registrar e gerenciar consultas médicas, pacientes e profissionais de saúde.
- Escopo: Backend em Node.js/TypeScript com banco de dados Prisma.

## Objetivos

- Fornecer uma API REST para cadastro e consulta de dados
- Melhorar a organização e manutenção do projeto
- Documentar o contexto de desenvolvimento dentro de `src`

## Estrutura do diretório `src`

- `backend/`: implementação do servidor e configuração do ambiente.
- `README.md`: instruções gerais de uso e setup.
- `contexto.md`: este arquivo de contexto do projeto.
